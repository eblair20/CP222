This directory contains the java source files for a data structure representing a point in 2D space.

Files:
IIntPoint2D.java - An interface for a 2D point with integer coordinates
Test.java        - A program to test an 2D point implementation
IntPoint2D.java  - A program that implements IIntpoint2D.java interface and redefines the methods to compute the values that match the ones desired in the Test.java file
